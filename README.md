# 🎬 KinoDOGandFOX

Cinema reference service — manage cinemas, films, sessions and generate reports.

---

## 📥 Installation

1. Go to **Releases** section
2. Download `Setup.rar`
3. Extract the archive
4. Run `setup.exe`
5. Follow the installer instructions

---

## Features

- Cinemas: add, edit, delete
- Films: add, edit, delete
- Sessions: add, edit, delete with date picker
- Cinema repertoire for selected date
- Free seats check for specific session
- Film sales statistics for selected period
- Save reports to txt file

---

## 🛠️ Tech Stack

C++/CLI, Windows Forms, .NET Framework 4.7.2
